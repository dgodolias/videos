graphify

Κάνε οποιονδήποτε φάκελο, κώδικα, docs ή PDF, έναν χάρτη που δείχνει πώς συνδέονται όλα. Μετά το Claude Code ρωτάει τον χάρτη αντί να ξαναδιαβάζει κάθε αρχείο, οπότε γλιτώνεις tokens. Είναι open source και τρέχει στον υπολογιστή σου.

commands 
# 1. Εγκατάσταση (μία φορά, στο terminal)
pip install graphifyy && graphify install

# 2. Φτιάξε τον χάρτη — μέσα στο Claude Code, στον φάκελό σου
/graphify .

# 3. Ρώτα το (το AI κοιτάζει τον χάρτη, δεν ξαναδιαβάζει τα αρχεία)
/graphify query "πώς συνδέεται το Χ με το Υ;"

Βήματα αναλυτικά

  1. Χρειάζεσαι Claude Code και Python 3.10+ στον υπολογιστή σου. Δεν χρειάζεσαι API key, χρησιμοποιεί τη σύνδεση που έχει ήδη το Claude.
  2. Άνοιξε ένα terminal και τρέξε μία φορά: pip install graphifyy && graphify install. Το package λέγεται graphifyy (με δύο y), η εντολή μένει graphify. Σε Windows, αν δεν το βρίσκει μετά, δοκίμασε pipx install graphifyy.
  3. Άνοιξε το Claude Code μέσα στον φάκελο ή το project που θες να καταλάβεις.
  4. Γράψε /graphify . για να φτιάξει τον χάρτη όλου του φακέλου. Σου δημιουργεί έναν φάκελο graphify-out/ με ένα διαδραστικό graph.html που το ανοίγεις και στον browser.
  5. Ρώτα ό,τι θες: /graphify query "η ερώτησή σου". Το AI απαντάει κοιτάζοντας τον χάρτη, με παραπομπή στο ακριβές αρχείο. Δες και /graphify path "Α" "Β" για τη διαδρομή ανάμεσα σε δύο πράγματα, και /graphify explain "Χ" για εξήγηση ενός κόμβου.
  6. Όταν αλλάξεις αρχεία, τρέξε /graphify . --update για να ανανεωθεί μόνο ό,τι άλλαξε. Με --watch ενημερώνεται αυτόματα καθώς δουλεύεις.
